GenerateUserToken Request¶

GET /GenerateUserToken/(str: user_token)¶

This get request generates an admin token for restbase-server.

Example request:

GET /GenerateUserToken HTTP/1.1
Host: example.com
Accept: application/json
Request Headers

  • admin_token – Admin token.

Query Parameters

  • user_token – Optional Pre-defined user token. If user_token is not defined it will be generated automatically.

Example response:

HTTP/1.1 200 OK
Vary: Accept
Content-Type: text/javascript

[
  {
      "status" : "success"
      "new_token": "some-user-token"
  }
]
Status Codes

  • 200 OK – No errors.

  • 409 Conflict – Token already exists.

  • 403 Forbidden – Access denied. Check admin token in a header.

  • 400 Bad Request – Bad request. Check a request body.
